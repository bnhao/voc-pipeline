# Voice of Client (VoC) Analysis Pipeline

## Project Overview
This project simulates the end-to-end processing of client feedback to improve Client Experience (CX) metrics. It focuses on maintaining a quality data foundation and scaling insights via automated reporting.

## Features
* **Data Transformation:** Automated cleaning of open-text feedback using Python/Pandas.
* **NLP Modeling:** Leverages **BERT-based architectures** for sentiment and category extraction.
* **Benchmarking:** Implements evaluation metrics (F1-Score, Precision) to monitor for model drift.
* **Generative AI:** Integrated with **Google Gemini** to generate automated executive summaries.

## Tech Stack
* **Language:** Python, SQL
* **Models:** Hugging Face (BERT), Google Gemini
